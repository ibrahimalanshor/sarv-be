# Sarv Server

Built with Adonis Js and MySQL.

## Instalation

Clone Repository.

```bash
git clone git@github.com:ibrahimalanshor/sarv-server.git
```

Instal dependencies.

```bash
npm install
```

Copy Environtment.

```bash
cp .env.example .env
```

## Dev Server

```bash
npm run dev
```

Open `http://localhost:3333`.

## Build Production

```bash
npm run build
```

## Production Server

```bash
npm start
```
